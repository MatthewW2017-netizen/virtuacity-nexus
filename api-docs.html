API Reference

Complete documentation for all Nexus OS APIs. Every endpoint documented with examples, SDKs, and best practices.

Getting Started

API Base URL

https://api.nexusos.com/v1

Authentication Header

Authorization: Bearer YOUR_API_KEY

Response Format

{ "success": true, "data": { ... }, "meta": { "timestamp": "2026-01-31T05:27:18Z", "version": "1.0" } }

User API

Manage user profiles, authentication, and identity.

GET
/users/{user_id}
Retrieve user profile information
curl -X GET https://api.nexusos.com/v1/users/user_123 \ -H "Authorization: Bearer YOUR_API_KEY" Response: { "success": true, "data": { "id": "user_123", "username": "john_doe", "email": "john@example.com", "created_at": "2026-01-15T10:30:00Z", "avatar_url": "https://...", "verified": true } }
POST
/users
Create a new user account
curl -X POST https://api.nexusos.com/v1/users \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "username": "jane_doe", "email": "jane@example.com", "password": "secure_password" }'
PUT
/users/{user_id}
Update user profile

Messaging API

Send and receive messages, manage conversations.

POST
/messages
Send a message to a space or user
curl -X POST https://api.nexusos.com/v1/messages \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "space_id": "space_456", "content": "Hello Nexus!", "type": "text", "mentions": ["user_789"] }'
GET
/spaces/{space_id}/messages
Fetch messages from a space with pagination
DELETE
/messages/{message_id}
Delete a message

Economy API

Manage virtual currencies, transactions, and monetization.

GET
/wallets/{user_id}
Get user wallet balance
Response: { "success": true, "data": { "user_id": "user_123", "currency": "NXS", "balance": 1500, "locked_balance": 0, "updated_at": "2026-01-31T05:27:18Z" } }
POST
/transactions
Create a transaction between users
curl -X POST https://api.nexusos.com/v1/transactions \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "from_user_id": "user_123", "to_user_id": "user_789", "amount": 100, "currency": "NXS", "description": "Purchase: Premium Badge" }'

Game API

Leaderboards, achievements, and multiplayer features.

POST
/leaderboards
Create a new leaderboard
curl -X POST https://api.nexusos.com/v1/leaderboards \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "name": "Weekly Scores", "type": "global", "period": "weekly", "order": "descending" }'
POST
/leaderboards/{leaderboard_id}/scores
Submit a score to leaderboard
GET
/leaderboards/{leaderboard_id}/top
Get top scores from leaderboard

Analytics API

Track events, analyze user behavior, and get insights.

POST
/events
Track a custom event
curl -X POST https://api.nexusos.com/v1/events \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "user_id": "user_123", "event_name": "purchase_completed", "properties": { "item_id": "item_456", "amount": 9.99, "currency": "USD" } }'
GET
/analytics/events
Query events with filters and aggregations

Notifications API

Send push notifications, emails, and webhooks.

POST
/notifications/send
Send notification to user
curl -X POST https://api.nexusos.com/v1/notifications/send \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "user_id": "user_123", "type": "push", "title": "Welcome!", "body": "Your account is ready", "data": { "action_url": "/dashboard" } }'
POST
/webhooks
Register a webhook endpoint

Error Handling

HTTP Status Codes

200 OK - Request successful

201 Created - Resource created successfully

400 Bad Request - Invalid parameters

401 Unauthorized - Missing or invalid API key

403 Forbidden - Permission denied

404 Not Found - Resource not found

429 Too Many Requests - Rate limit exceeded

500 Internal Server Error - Server error

Error Response Format

{ "success": false, "error": { "code": "INVALID_REQUEST", "message": "Missing required field: email", "details": { "field": "email", "reason": "required" } } }

Rate Limiting

Free tier: 100,000 requests/month (3.3 per second)

Professional: 5,000,000 requests/month (165 per second)

Enterprise: Custom limits with dedicated support

Rate Limit Headers

X-RateLimit-Limit: 3600 X-RateLimit-Remaining: 3599 X-RateLimit-Reset: 1643619600
