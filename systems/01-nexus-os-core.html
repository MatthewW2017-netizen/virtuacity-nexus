Nexus OS Core

The base operating system — modes, panels, identity, notifications, membership, and verification.

Overview

Nexus OS Core is the kernel of the platform: it defines how Modes behave, how Panels render, how identities are scoped, and how membership and verification are enforced. It is lightweight, extensible, and designed to run in the browser as the authoritative surface of the OS.

  • Mode engine: contextual behavior and state
  • Panel system: dockable UI surfaces
  • Identity framework: verified IDs, RP/Studio/Creator personas

How it fits in Nexus OS

All other systems are modules built atop the Nexus OS Core. City OS, RP OS, Studio OS, and expansion modules communicate through Core contracts: identities, events, notifications, and persistent state.
