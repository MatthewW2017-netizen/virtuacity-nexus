Server Guide System

Multi-guide engine for City Guide, RP Guide, Studio Guide, Creator Guide, Bot Guide, and Mini‑App Guide.

Overview

The Server Guide System centralizes onboarding, rules, how‑tos, and city policies. Guides are composable, versioned, and can be scoped to Cities, Modes, or Spaces. They are the authoritative documentation surfaced to users during onboarding and while using OS panels.

  • Composable, versioned guides
  • City-scoped rules and onboarding
  • Integration with appeals and Security OS

How it fits in Nexus OS

Guides provide context-aware help across the OS, feeding into Customization, Appeals, and Analytics to ensure consistent governance and better onboarding outcomes.
