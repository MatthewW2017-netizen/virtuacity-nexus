Chat System

Identity-driven chat with Reply Capsules, seen indicators, mode-based reactions, and message-triggered workflows.

Overview

Nexus Chat is identity-first: messages carry RP icons, named personas, auras, and mode tags. Reply Capsules present contextually rich replies; seen indicators respect privacy and status; mode-based reactions change behavior depending on Studio or RP contexts. Messages can also trigger automations, tasks, and logs across systems.

  • Persona-aware messages
  • Reply Capsules & seen indicators
  • Message-triggered workflows

How it fits in Nexus OS

Chat acts as an event bus for human interaction—feeding Automation, Analytics, RP logs, and Studio tasks while reflecting identity and mode context provided by Core and Profile systems.
